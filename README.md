# 🐎 競馬スコアラー - Django採点システム

Excelファイルをアップロードするだけで、
自動採点・レポート生成・結果可視化を行う Django Webアプリです。

---

# 📸 アプリ概要

正解マスタExcelとユーザー解答Excelを読み込み、

- 自動採点
- 正答率算出
- ランク判定
- HTMLプレビュー
- Excelレポート出力
- フィードバック動画表示

を行う学習支援アプリです。

---

# 🛠 開発背景

私はPython資格試験の取得に向けて、過去問題集や模擬試験を使った自己学習を行っていました。

しかし、毎回PDFや解答資料を見ながら手作業で自己採点を行う必要があり、

学習支援ツールとして、本アプリを開発しました。

■課題
- 採点に時間がかかる
- ミスが発生しやすい
- 学習結果を蓄積しづらい
- 苦手分野を把握しにくい

☑実現
- Excelファイルをアップロードするだけで自動採点
- 正答率やランクを即時表示
- 結果をExcelレポートとして保存
- 苦手分野を可視化して復習効率を向上
- 視覚的なフィードバックによる学習モチベーション向上

単なる採点ツールではなく、

「学習効率の向上」と「継続的な学習モチベーションの維持」

を目的として設計しています。

また、実装を通して、

- DjangoによるWebアプリ開発
- pandasを用いたデータ処理
- openpyxlによるExcelレポート生成
- Tailwind CSSによるUI設計

など、実務を意識した技術構成にも取り組みました。

---
